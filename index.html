🔄 Migración de Datos
Cómo mover tus datos entre dispositivos sin perder nada
💾 Exportar datos (PC / Tablet → Móvil)
1Ve a Ajustes → Backup → Exportar JSON en el dispositivo de origen.
2El archivo se llama backup_pedidos_DD-M-YYYY.json. Guárdalo en la nube (Drive, WhatsApp, email) o pásalo al otro dispositivo.
3En el dispositivo destino, abre Franez y ve a Ajustes → Backup → Importar JSON.
4Selecciona el archivo JSON. Atención: esto sobreescribe todos los datos del dispositivo destino.
⚠ Qué sobreescribe y qué no
Sobreescribe completamente: clientes, productos, ofertas, historial, stock, campañas, configuración.
No toca: pedidos activos en curso (los 3 slots) — estos son locales del dispositivo y no se exportan.
Recomendación: exporta siempre después de guardar el último pedido en historial, no antes.
📷 Fusionar backups (varios dispositivos)
Si has estado trabajando en varios dispositivos a la vez y cada uno tiene pedidos distintos, los backups se pueden fusionar. Descarga los JSON de cada dispositivo y fusiónalos con ayuda de Claude — sube los archivos y pide: "Fusiónanos estos backups de Franez".
✓ La fusión respeta todos los pedidos de todos los dispositivos, elimina duplicados y ordena por fecha.
📱 Instalar Franez en el móvil
Franez es una PWA — funciona como app instalada sin tienda de apps:
Android:Abre el HTML en Chrome → menú (⋮) → Añadir a pantalla de inicio
iPhone:Abre el HTML en Safari → botón compartir → Añadir a pantalla de inicio
